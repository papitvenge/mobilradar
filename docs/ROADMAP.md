# NorScan – Roadmap

## Fase 0 (nå)

- Prosjekt rebrand til NorScan.
- Definert produktmål og arkitektur.
- MVP-UI for skanning, kalibrering, tegning og eksportvalg.

## Fase 1

- Native LiDAR-inntak på iOS.
- Første fotoskanning-pipeline på Android.
- Grunnleggende lagring av skanningssesjoner.

## Fase 2

- Forbedret kalibrering med flere kontrollmål.
- 2D-tegningsgenerator med målsetting og eksport til PDF/DXF.
- 3D-visning med mål og snitt.

## Fase 3

- Produksjonsklare eksportmoduler (STL/OBJ/STEP/USDZ).
- Kvalitetsrapport per skanning (toleranse, avvik, confidence).
- Team-funksjoner og skybasert synkronisering.
